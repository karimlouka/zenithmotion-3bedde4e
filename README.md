# Zenith Motion

Advanced AI-powered workforce monitoring system. Track multiple workers in real-time, detect activity patterns, and receive instant alerts for inactivity.

## Features

- **Real-time Detection**: High-performance AI detection.
- **Multi-person Tracking**: Monitor multiple individuals simultaneously.
- **Smart Alerts**: Instant notifications for inactivity.
- **Multilingual Support**: Available in English, French, and Arabic.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **AI**: TensorFlow.js (COCO-SSD)
- **UI Components**: Radix UI, Lucide React
